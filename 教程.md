**1. 准备目录**

```
mkdir D:\AI\other_project
cd D:\AI\other_project
git clone https://github.com/calesthio/OpenMontage.git
cd OpenMontage
```

**2. 检查 Git、Node、npm**

```
git --version
node --version
npm --version
```

能输出版本号就行。Node 建议 LTS 版。

**3. 安装 Python 3.12**

如果已经有 Python 3.14，也没关系，额外装 3.12 不冲突。

```
winget install --id Python.Python.3.12 -e
```

装完重新打开 PowerShell，检查：

```
py -0p
py -3.12 --version
```

能看到 `Python 3.12.x` 就行。

**4. 给 OpenMontage 创建独立虚拟环境**

```
cd D:\AI\other_project\OpenMontage
py -3.12 -m venv .venv
.\.venv\Scripts\python.exe --version
```

确认输出是 `Python 3.12.x`。

**5. 安装 Python 依赖**

```
.\.venv\Scripts\python.exe -m pip install --upgrade pip
.\.venv\Scripts\python.exe -m pip install -r requirements.txt
```

**6. 安装 Remotion 依赖**

```
cd remotion-composer
npm install
cd ..
```

如果 `npm install` 报 `ERR_INVALID_ARG_TYPE`，换成：

```
cd remotion-composer
npx --yes npm install
cd ..
```

**7. 全局安装 FFmpeg**

```
winget install --id Gyan.FFmpeg -e
```

装完重新打开 PowerShell，检查：

```
ffmpeg -version
ffprobe -version
```

两个都能输出版本号才算 OK。

**8. 创建环境变量文件**

```
copy .env.example .env
```

先别急着配本地 GPU。第一轮测试建议不打开：

```
VIDEO_GEN_LOCAL_ENABLED=true
```

也就是先不要配这项。

**9. 第一次测试建议**

先用免费素材/本地合成/API 路线，不要一上来跑本地视频模型。

可以在 Codex / Claude Code / Cursor 里打开：

```
D:\AI\other_project\OpenMontage
```

然后输入：

```
帮我做一个 30 秒中文解说视频，主题是「AI Agent 为什么正在改变视频制作」。
优先使用免费素材和本地合成。
需要中文旁白、字幕、背景音乐，最后导出 MP4。
成本控制在 1 美元以内。
```

**核心原则**

Python 相关命令都用：

```
.\.venv\Scripts\python.exe
```

不要直接用：

```
python
pip
```

FFmpeg 用全局命令：

```
ffmpeg
ffprobe
```

这样路径最清楚，也最适合写成公众号教程。
